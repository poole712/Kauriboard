# KauriBoard

**Project Description:** KauriBoard is the ultimate project planning tool. KauriBoard fulfills all project teams needs by offering them a place for planning and collaboration with live updates and interactions. 

**Stack:**
- Frontend:
  - React (Vite)
  - Typescript
  - React Query
  - React Router
  - Tailwind

- Backend:
  - .NET 8 Web API
  - Entity Framework Core
  - SQL Server or PostgresSQL
  - JWT authentication
  - SignalR
 
- Infrastructure
  - Azure App Service (API)
  - Azure SQL
  - Vercel (Frontend)
